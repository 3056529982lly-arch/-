# Amazon Operations Onboarding Dashboard

A clean Next.js training dashboard for onboarding new Amazon operations staff.

## Pages

- Dashboard
- Training Roadmap
- Amazon Modules
- SOP Library
- Practice Tasks
- Quiz & Assessment

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn/ui-style components
- lucide-react icons

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
